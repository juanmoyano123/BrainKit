## Validaciones

### Email
| Validacion | Mensaje | Cuando |
|------------|---------|--------|
| Required | "Email is required" | On submit |
| Format | "Please enter a valid email" | On blur |
| Exists (login) | "No account with this email" | On submit |
| Exists (register) | "Account already exists. Sign in?" | On submit |

### Password
| Validacion | Mensaje | Cuando |
|------------|---------|--------|
| Required | "Password is required" | On submit |
| Min length | "At least 8 characters" | On type |
| Uppercase | "Include 1 uppercase letter" | On type |
| Number | "Include 1 number" | On type |
| Match | "Passwords don't match" | On blur (confirm) |

### Strength Indicator
```
Weak:   < 8 characters
        [==                    ] Red

Medium: 8+ chars
        [=========             ] Yellow

Strong: 8+ chars + number + uppercase
        [====================] Green
```

## Interacciones

| Elemento | Accion | Resultado |
|----------|--------|-----------|
| Google OAuth | Click | Open Google popup |
| Email input | Blur | Validate format |
| Password input | Type | Update strength |
| Eye icon | Click | Toggle password visibility |
| Submit | Click | Validate all, submit |
| Link (sign in) | Click | Navigate to login |
| Link (create) | Click | Navigate to register |
| Forgot password | Click | Navigate to forgot |
