## 2. Competidores Analizados (2026)

### 2.1 Duolingo
**Paleta:** Verde (#58CC02), complementos en amarillo/rojo/azul
**Fortalezas de Progress Tracking:**
- Streaks prominentes (aumentan engagement 60%)
- XP system con barras de progreso visuales
- Ligas semanales para competencia social
- "Corazones" como sistema de vidas
- Celebraciones animadas al completar lecciones

**Metricas clave:**
- 55% retention rate gracias a gamificacion
- Usuarios 2.3x mas propensos a engagement diario con streak 7+ dias

**Insight para BrainKit:** El verde transmite crecimiento y exito. Streaks son esenciales.

---

### 2.2 Strava
**Paleta:** Naranja coral (#FC4C02), negro, blanco
**Fortalezas de Progress Tracking:**
- Activity heatmaps (mapas de calor de actividad)
- Weekly/monthly/yearly summary cards
- Personal records (PRs) con badges
- Progress rings estilo Apple Watch
- Comparativas con semanas anteriores

**Insight para BrainKit:** El naranja coral transmite energia y movimiento. Los heatmaps son poderosos para visualizar consistencia.

---

### 2.3 Headspace
**Paleta:** Naranja calido (#F47D31), ilustraciones organicas
**Fortalezas de UX:**
- Interfaz calma y espaciada
- Progress tracking sutil (no invasivo)
- Streaks de meditacion sin presion
- Colores que reducen ansiedad

**Insight para BrainKit:** Para profesionales de salud, colores calidos + espacio generoso = confianza.

---

### 2.4 Calm
**Paleta:** Azules profundos y verdes bosque (#0A2239, verde esmeralda)
**Fortalezas:**
- Paleta naturaleza = confianza
- Animaciones lentas y fluidas
- Modo oscuro premium

**Insight para BrainKit:** El verde esmeralda es profesional y distintivo.

---

### 2.5 Notion (2025-2026)
**Paleta:** Neutrales con acentos minimos
**Fortalezas de Dashboard:**
- Limpios, funcionales
- Progress bars en bases de datos
- Charts integrados
- Gamificacion a traves de templates (habit trackers)

**Insight para BrainKit:** Simplicidad + progress bars inline = profesionalismo.

---

### 2.6 Todoist (Karma System)
**Paleta:** Rojo coral (#E44332)
**Fortalezas de Progress:**
- Karma points system
- Daily/weekly goals
- Productivity trends graphs
- Streaks por proyectos completados

**Insight para BrainKit:** El rojo coral es energetico pero profesional.

---

### 2.7 Forest App
**Paleta:** Verde naturaleza, marron tierra
**Fortalezas:**
- Metafora visual (arbol crece con focus)
- Gamificacion con proposito (plantar arboles reales)
- Progress = bosque visual

**Insight para BrainKit:** El verde naturaleza transmite crecimiento y bienestar.

## 4. Analisis de Progress Tracking UX

### Patrones Observados en Apps Exitosas (2025-2026)

#### 4.1 Streaks
| App | Implementacion | Efectividad |
|-----|---------------|-------------|
| Duolingo | Icono fuego + contador prominente | +60% engagement |
| Headspace | Sutil, sin presion | +35% weekly active |
| Todoist | Por proyecto completado | +25% task completion |
| Strava | Activity calendar | +40% return rate |

**Best Practice:** Streak visible pero no culpabilizador

#### 4.2 Progress Rings/Bars
| App | Implementacion |
|-----|---------------|
| Apple Fitness | 3 rings concentricos (Move, Exercise, Stand) |
| Duolingo | Barra lineal con XP |
| Notion | Barras en databases |
| Todoist | Circular con % completado |

**Best Practice:** Rings para metricas multiples, bars para progreso lineal

#### 4.3 Heatmaps de Actividad
| App | Implementacion |
|-----|---------------|
| GitHub | Calendar heatmap anual |
| Strava | Activity calendar con intensidad |
| Obsidian | Plugin contribution graph |

**Best Practice:** Colores de menor a mayor intensidad (light green to dark green o custom)

#### 4.4 Estadisticas y Analiticas
| Metrica | Visualizacion recomendada |
|---------|--------------------------|
| Cards studied today | Big number + barra vs goal |
| Streak actual | Badge con fuego + contador |
| Retention rate | % grande + trend arrow |
| Cards due | Badge numerico en deck card |
| Study time | Timer + weekly total |
| Mastery level | Progress ring % |

## 5. Recomendacion de Paleta para BrainKit V2

### Opcion Seleccionada: Emerald + Amber (Natural Premium)

**Justificacion:**
1. **Verde Esmeralda** como primario:
   - Transmite: Crecimiento, conocimiento, confianza
   - Diferente a: Quizlet (azul), Duolingo (verde lima)
   - Profesional para: Enfermeras, medicos, abogados
   - Asociacion: Naturaleza, aprendizaje, progreso

2. **Ambar/Oro** como acento:
   - Transmite: Logro, premium, exito
   - Usado para: Streaks, achievements, highlights
   - Complementa verde sin competir

3. **Gradientes naturales:**
   - From: Esmeralda (#059669)
   - To: Teal (#0D9488)
   - Resultado: Moderno, distintivo, profesional

## 6. Sistema de Progress Tracking Propuesto

### Componentes Nuevos Requeridos

#### 6.1 StreakBadge
- Icono de fuego en amber (#F59E0B)
- Contador de dias
- Animacion sutil al incrementar
- Estado "freeze" si se pierde (gris)

#### 6.2 ProgressRing
- SVG circular con porcentaje
- Colores: primary gradient
- Tamanios: sm (40px), md (80px), lg (120px)
- Label central con % o numero

#### 6.3 StatCard
- Numero grande (metric)
- Label pequeno
- Trend indicator (arrow up/down)
- Colores segun contexto

#### 6.4 ActivityHeatmap
- Grid calendar estilo GitHub
- 5 niveles de intensidad (0-4)
- Colores: primary-100 to primary-700
- Tooltip con fecha y count

#### 6.5 WeeklyProgressBar
- 7 segmentos (Lun-Dom)
- Fill segun cards estudiadas
- Goal line indicador
- Hover con tooltip

### Ubicaciones en UI

| Componente | Dashboard | Deck Detail | Study Session | Stats Page |
|------------|-----------|-------------|---------------|------------|
| StreakBadge | Header, Hero | - | Header | Hero |
| ProgressRing | Per-deck cards | Mastery % | - | Per-metric |
| StatCard | Weekly summary | Deck stats | - | All stats |
| ActivityHeatmap | Mini (last 7 days) | - | - | Full year |
| WeeklyProgressBar | Hero section | - | - | Weekly detail |
| SessionProgress | - | - | Top bar | - |

## 7. Conclusiones y Proximos Pasos
